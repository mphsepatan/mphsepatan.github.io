Cara Donasi:

  1. Pilih nominal donasi atau masukkan jumlah sendiri
  2. Klik tombol "Donasi via WhatsApp"
  3. Anda akan diarahkan ke chat WhatsApp dengan admin
  4. Kirim konfirmasi donasi beserta bukti transfer (jika transfer)
  5. Admin akan mencatat donasi Anda di sistem
  6. Nama Anda akan muncul di daftar donatur setelah diproses

Informasi Penting: Semua donasi akan dicatat secara transparan dan dapat dilihat di halaman Donatur. Admin akan mengupdate data maksimal 24 jam setelah donasi diterima.
